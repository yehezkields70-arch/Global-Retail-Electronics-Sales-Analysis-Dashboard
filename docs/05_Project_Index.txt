================================================================================
                       QUICK START - LANGKAH KERJA
================================================================================

STEP 1: BACA DOKUMENTASI
------------------------
1. Baca README.md untuk memahami project secara keseluruhan
2. Baca Project Brief untuk konteks bisnis

STEP 2: PERSIAPAN DATA
----------------------
1. Siapkan file data: Sales.csv, Products.csv, Stores.csv, Customers.csv
2. Buka Power BI Desktop
3. Load data menggunakan "Get Data"

STEP 3: DATA TRANSFORMATION
---------------------------
1. Buka Power Query Editor
2. Ikuti panduan di: 02_PowerQuery_Transformation_Guide.txt
3. Lakukan semua transformasi yang diperlukan
4. Buat Date Table
5. Close & Apply

STEP 4: DATA MODELING
---------------------
1. Buka Model View
2. Buat relasi sesuai Star Schema:
   - Sales → Products (Many-to-One)
   - Sales → Stores (Many-to-One)
   - Sales → Customers (Many-to-One)
   - Sales → Date Table (Many-to-One)

STEP 5: DAX MEASURES
--------------------
1. Buka 03_DAX_Measures_Documentation.txt
2. Copy-paste setiap measure ke Power BI
3. Verifikasi hasil perhitungan

STEP 6: BUILD DASHBOARD
-----------------------
1. Buka 04_Dashboard_Layout_Guide.txt
2. Buat visualisasi sesuai panduan:
   - Row 1: 4 KPI Cards
   - Row 2: Line Chart + Map
   - Row 3: Donut Chart + Horizontal Bar
   - Row 4: Clustered Bar + Donut
   - Row 5: Scatter Plot
   - Slicers: Tahun, Negara, Kategori, Channel

STEP 7: FORMATTING & POLISH
---------------------------
1. Atur tema dan warna
2. Alignment dan spacing
3. Tambahkan tooltips
4. Test interaktivitas slicers

STEP 8: PUBLISH
---------------
1. Save file .pbix
2. Publish ke Power BI Service
3. Buat App untuk sharing
4. Tambahkan ke portofolio!
